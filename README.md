# Nixon Auto Parts

Enterprise SaaS platform for buying, selling, and managing auto parts.

## Features

- Multi-tenant SaaS
- Secure authentication
- Inventory management
- Orders and quotations
- Customer management
- Supplier management
- Payments
- Analytics dashboard
- AI-powered insights
- Real-time notifications
- Billing and subscriptions
- Admin control panel

## Technology

- Next.js
- TypeScript
- Prisma
- PostgreSQL
- Tailwind CSS
- NextAuth
- Stripe
- Vercel

## Deployment

This application is designed for deployment on Vercel.
